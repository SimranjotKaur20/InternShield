About InternShield

InternShield is an AI-powered cybersecurity platform designed to protect students from fake internship offers, phishing domains, and payment scams. Our system analyzes internship messages, detects suspicious patterns, checks domain intelligence, and guides users to report cybercrime safely.

Key Features

  • 🔍 Internship Scam Detection
  • 🌐 Domain Intelligence & Reputation Check
  • 📄 PDF Security Report Generation
  • ⚠ Cyber Scam Awareness Resources
  • 🚨 Direct Reporting to Government Cyber Crime Portal

How InternShield Works

  1. User submits internship message or link.
  2. InternShield scans for scam keywords and suspicious patterns.
  3. Domain intelligence APIs analyze the website reputation.
  4. The system generates a risk score and detailed report.
  5. If suspicious, users can report directly to the Cyber Crime Portal.

Technology Stack

Frontend: HTML, CSS, JavaScript

Backend: Python Flask Security Tools: WHOIS, Domain Reputation API Libraries: PyPDF2, Requests, ReportLab

Our Mission

To create a safer digital environment for students by preventing internship fraud and empowering users with cybersecurity awareness.
